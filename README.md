# ActiveGear Data Analyst Job Simulation

This repository simulates a junior Data Analyst workflow using Python, Pandas and NumPy.

## Fictional Company

**ActiveGear Retail** is a sports retail company operating across Latin America. The business sells footwear, apparel, accessories and training equipment through physical stores and online channels.

## Goal

Practice realistic data analysis tasks using business tickets, shared datasets and clear deliverables.

## Recommended Workflow

1. Read the ticket description.
2. Inspect the required CSV files.
3. Write your solution in `solution.py` or `solution.ipynb`.
4. Save the requested output file inside the ticket `output/` folder.
5. Commit your work to Git with a clear message.

## Skills Practiced

- Reading CSV files
- Data exploration
- Filtering rows
- Creating calculated columns
- Handling missing values
- Removing duplicates
- Merging datasets
- Grouping and aggregating data
- Exporting clean reports
